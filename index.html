HTML

HTML means hypertext markup language

About HTML

  • It is the standard markup language for creating webpages and web applications
  • It defines how content like text,images,videos and links appear in a browser
  • HTML elements are the basic building blocks of a webpage defining its structure
  • HTML elements start with an opening tag and end with closing tag
  • some elememts are self closing and browsers use elements to render the page visually



CSS

CSS means Cascading style sheets

About CSS

  • CSS (Cascading Style Sheets) is a style sheet language
  • It is used to describe how documents written in a markup language
  • It is one of the three core technologies of the World Wide Web
  • CSS allows developers to separate a website's content from its design.
  • This model consists of four parts: Content,Padding,Border and Margin


JS

JS means Java Script

About JS

  • JavaScript (JS) is a lightweight, interpreted programming language
  • JavaScriptserves as the "engine" of the modern web.
  • JavaScript brings websites to life by making them interactive and dynamic.
  • Code is typically translated and executed line-by-line in the browser.
  • It supports various programming styles, including Functional and Object-Oriented programming.
